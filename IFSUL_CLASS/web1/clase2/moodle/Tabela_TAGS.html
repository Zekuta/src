Tabla: Semántica de HTML

TAGS
DOCTYPE Es una instrucción especial que indica al navegador cual es la version de HTML que esta usando en el momento. En tal caso, informa que el documento es un archivo HTML5.
br Es un tag para quebrar la linea, actua como un "ENTER".
head Es un tag que almacena el título de una pagina y define la formatación de texto.
body Es un tag que almacena toda la codificacion de dentro de la webpage.
h1 Es un tag que define el titulo de la pagina web, puede ir desde h1 a h6, pero se aconseja que solamente haya un h1 en la pagina, segun las reglamentaciones de Google.
title Este tag define el titulo de la pagina web en la pestaña del navegador.
ul Es un tag que se utiliza para la creacion de listas sin un orden especifico.
ol Es un tag que se utiliza para la creacion de listas con un orden creciente.
meta Es una tag para definir la fuente de texto que se utilizara en la webpage.
li Es una tag que sirve para declarar todos los elementos de una lista
audio Es una tag para poder poner audio en la webpage como un mediaplayer.
hr Es una tag que sirve para crear una linea que ocupa todo el largo de la pantalla, si se lo pone dentro de un label, ocupara el espacio maximo dentro de ese label.
html Es la tag principal que indica cuando inicia y cuando cierra el programa.
img Es una tag que permite incluir imagenes dentro de la webpage
a Es una tag que sirve para crear links a partir de textos
p Es una tag que sirve para escribir parrafos tramos de texto, inserta el "ENTER" automatico.
figure Es una tag que sirve para marcar una imagen
figcaption Es una tag que sirve para poner un titulo a una imagen
tbody Es una tag que sirve para declarar un grupo contenido de el cuerpo de unatabla de html
thead Es una tag que sirve para declarar un grupo de cabeceras que contiene una tabla de html
label Es una tag que sirve para declarar una etiqueta para un elemento de una interfaz de usuario
forms Es una tag que sirve para almacenar el input de un usuario
input Es una tag que especifica campos donde el usuario puede inserir datos
option Es una tag que sirve para representar un item dentro de un select
select Es una tag que representa un control que muestra un menu de opciones
button Es una tag que sirve para crear y definir un boton en html
table Es una tag que define la creacion de una tabla en html
tr Es una tag que define una fila de celdas en una tabla
td Es una tag que define la celda de una tabla que contiene datos
